## Features

- Province filter and city search on the home page.
- Direct city URLs for sharing a specific city map.
- Interactive city maps with line filters, station search, and station labels.
- Four basemap styles:
  - Gaode / AutoNavi muted
  - Gaode / AutoNavi colorful
  - OSM muted
  - OSM colorful
- Leaflet is vendored locally to avoid depending on an external CDN at runtime.
- WGS84 coordinates are used for OSM basemaps.
- GCJ-02 coordinates are used for Gaode / AutoNavi basemaps.
